# Stabilna informatika | DevOps & System Administration

A single-page portfolio website for Stabilna informatika d.o.o., showcasing DevOps and System Administration services.

## Features

- **Responsive Design**: Built with Tailwind CSS for a seamless experience on all devices.
- **Terminal Theme**: A unique, interactive terminal-style interface.
- **Chaos Mode**: Click the logo 5 times to trigger a "Kernel Panic" simulation.
- **Anti-Spam**: Email address is obfuscated and loaded via JavaScript to prevent scraping.

## Technologies

- **HTML5**: Semantic markup.
- **Tailwind CSS**: Utility-first CSS framework (loaded via CDN).
- **Vanilla JavaScript**: Lightweight interactions without heavy frameworks.
- **Lucide Icons**: Beautiful & consistent icons.

## Setup & Usage

Since this project uses CDN links for dependencies, no build process is required.

1. Clone the repository.
2. Open `index.html` in your preferred web browser.
